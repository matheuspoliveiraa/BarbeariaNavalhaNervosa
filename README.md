💈 Barbearia Navalha Nervosa

Sistema simples de gerenciamento de serviços para barbearia, com interface gráfica intuitiva, desenvolvido para facilitar o cadastro, busca e organização de serviços.

📸 Preview

Interface com tema visual inspirado em barbearias modernas, incluindo ícones interativos e layout limpo.

## 🚀 Funcionalidades

- 🔍 Buscar serviço pelo ID  
- ➕ Cadastrar novos serviços  
- ✏️ Editar serviços existentes  
- 🗑️ Excluir serviços  
- 💲 Inserir valores formatados  
- 🖼️ Carregar imagem do serviço  
- 📋 Interface visual com botões ilustrativos  
- 📅 Exibição de data atual no sistema
